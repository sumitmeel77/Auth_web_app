Logout Password Change

This web app uses following frameworks - Nodejs , express , mongodb ,javascript , html , css .
So basically this app is authentication app that allows user to login , signup and password change. So at home page there are two buttons one for sign in and other for register . So when user registers its data is stored in mongodb and user is redirecterd to login page.
After user login a token is generated using jwt(jsonwebtoken).and the token is stored with user data in mongodb. Also token is stored in cookies.
One's logined user is directed to main page with password change and logout button. So when user log out the app the cookie and token coreesponding to current device is deleted. And when login user change password then all token stored of differnet devices gets deleted and current user si redirected to home page . So when a registerd user login again after logout or password change then its stored cookies data is checked in database . if it is present then it is redirected to Home page else not.
